Tic Tac Toe

A Tic Tac Toe game seemed like a great project for me to get a better hand on experience using Javascript. This project gave me a deeper understanding on the fundamentals of programming. My approach to this game was to create a 3x3 matrix that would have specific coordinates that I could reference too. The user starts the game by clicking the squares which initiates the computers turn. The computer then scans through all the squares knowing what's available for a possible turn. Once it has fully scanned the arena it randomly guesses a coordinate and takes it's turn.

From there the game continues between the user and the computer until someone has finally won, loss, or a draw. Which forces the game to restart and the user can continue to play again.

The biggest challenge of this project was to create a computer that had the knowledge of knowing when to take it's turn, which square to pick, and to know when the user or computer has won. But the challenge gave me a opportunity to learn a lot of features of the Javascript language. The improvements I would like to make to this project will be a scoreboard, different difficulties for the computer, and possibly in the future to make a matchmaking setting.

Full Build with custom menu

I was given the opportunity to create a custom hamburger menu for a new theme we we're building at my job. This project definitely had it's challenges which were, how will our users interact with this menu, if our users have multiple navigational items create dropdowns, how will the user interact with the dropdowns, and making sure the menu worked on multiple browsers and devices.

Also making the navigation a full width menu gave me a lot of real estate to play around with layout and design. I got a chance to work closely with our UI/UX lead to understand the best practices for the hamburger menu. Which were hamburger menu placement, how would our client age group interact with the menu, and what modern styling could we add to the menu without causing to big of a distraction from the main functionality.
